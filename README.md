# 🎬 Marvel Cinematic Universe Movies Analysis Dashboard

A fully interactive and visually immersive Power BI dashboard analyzing the complete Marvel Cinematic Universe (MCU) movie saga through financial performance, ratings, characters, powers, and cinematic trends.

## 📌 Project Overview

This project presents a comprehensive analysis of MCU movies from Phase 1 to Phase 6 using advanced data visualization and dashboard storytelling techniques in Power BI.

The dashboard explores:

- 📈 Box office revenue trends
- 🎭 Character appearances & affiliations
- 💰 Budget vs profit analysis
- ⭐ IMDb & Rotten Tomatoes ratings
- 🦸 Hero, villain & antihero distributions
- ⚡ Powers and abilities insights
- 🎬 Movie-wise performance metrics

Designed with a cinematic Marvel-inspired UI theme for an engaging analytical experience.

## 📈 Key Insights

- Avengers-related movies generated the highest global revenues.
- Phase 3 dominated the MCU financially and critically.
- Hero characters significantly outnumber villains.
- Superhuman strength is the most common MCU power.
- Higher production budgets generally led to larger box office collections.
